# Make it a Quote
## 使い方
- `pip install -r requirements.txt`を実行
- `python main.py`で起動
- `http://localhost:3000/`でアクセスする
## パラメーター
- name: 名前
- tag: 4桁のID
- id: ID
- content: 内容
- icon: アイコン
