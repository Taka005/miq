# Make it a Quote
## 使い方
- `pip install -r requirements.txt`を実行
- `python main.py`で起動
- `http://localhost:3000/`でアクセスする
## エンドポイント
- `/`で生成
## パラメーター
- type: タイプ
- - `color` カラーアイコン
- - `reverse` アイコンの位置反転
- - `reverseColor` カラーアイコン・アイコンの位置反転
- - `white` 色反転
- - `reverseWhite` 色反転・アイコンの位置反転
- name: 名前
- id: ID
- content: 内容
- icon: アイコン(URL)
